## 문제점 (v4.5)
- [x] v4.5에서 PIKIT 곡괭이 scale=0.5로 축소 → 블록이 한 줄로만 파괴되는 문제
- [x] 원인 분석: 작은 hitbox → 좁은 범위에서만 블록 접촉

## 해결 방식
- [x] 시스템 곡괭이 scale=1.5 고정 (큰 크기 유지)
- [x] gravityMult 0.7→0.3 (매우 느리게 떨어짐 — 떠다니는 느낌)
- [x] speedMult 0.55→0.1 (수평 이동 극소 — 한 곳에 머무름)
- [x] damage 6→5 (약한 데미지로 블록 파괴 속도 감소)
- [x] 결과: 큰 곡괭이가 천천히 떠다니며 간간이 블록 파괴 → encounter rate 0.65/s
