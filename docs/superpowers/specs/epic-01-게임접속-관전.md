# [Epic 1] 게임 접속 및 관전 서비스
> 작성 기준: v5.0 | 작성일: 2026-03-24
> 관점: 유저 플로우 (비개발자용)

## 목적
비로그인 유저가 별도 가입 없이 즉시 게임 화면을 관전할 수 있도록 하며, 실시간 HUD 정보와 알림 시스템을 통해 몰입감 있는 관전 경험을 제공한다.

---

### [STORY 1-1] 유저는 처음 접속하면 로그인 없이 게임 화면을 즉시 볼 수 있어야 한다.
1. 유저가 브라우저에서 PIKIT 주소에 접속한다.
2. 게임 화면(캔버스)이 로그인 없이 즉시 표시된다.
3. 곡괭이가 낙하하고 블록이 파괴되는 실시간 장면이 나타난다.
4. 다른 플레이어들의 활동도 실시간으로 함께 표시된다.
5. 유저는 별도의 가입이나 로그인 없이 게임을 관전할 수 있다.

---

### [STORY 1-2] 유저는 화면 상단 HUD에서 실시간 정보를 확인할 수 있어야 한다.
1. 화면 상단에 정보 표시줄(HUD)이 항상 노출된다.
2. **채굴 깊이**: 현재 게임 진행 깊이가 미터(m) 단위로 실시간 갱신된다.
3. **크레딧 잔액**: 상단 우측에 현재 보유 크레딧이 표시된다. 비로그인 시 0으로 표시된다.
4. **필드 전환 스위치**: Normal(1x) / Hardcore(10x) 필드를 전환할 수 있는 스위치가 표시된다.
5. 모든 수치는 서버 상태에 따라 자동으로 실시간 갱신된다.

---

### [STORY 1-3] 유저는 메뉴 버튼을 클릭해 게임 정보와 블록 가이드를 볼 수 있어야 한다.
1. 유저가 화면 상단 좌측의 메뉴 버튼(햄버거 아이콘)을 클릭한다.
2. 화면 왼쪽에서 메뉴 패널이 슬라이드되어 나타나며, 배경이 반투명하게 어두워진다.
3. 패널에는 두 가지 정보가 표시된다:
   - **게임 현황**: 접속 중인 플레이어 수, 낙하 중인 곡괭이 총 개수
   - **블록 가이드**: 게임에 등장하는 블록 11종의 이름, 보상(크레딧), 등장 확률(%) 목록
4. 블록 가이드는 보상이 높은 순서로 나열된다 (Jackpot > Diamond > Netherite > Gold > ... > Dirt).
5. 각 블록 항목에는 블록 이미지, 이름, 보상 크레딧, 등장 확률이 함께 표시된다.
6. 패널 바깥의 어두운 영역 클릭 또는 X 버튼 클릭으로 패널이 닫힌다.

---

### [STORY 1-4] 유저는 희귀 블록 스폰 시 알림 배너를 볼 수 있어야 한다.
1. 게임 필드에 Diamond, Netherite, 또는 Jackpot 블록이 새로 등장한다.
2. 같은 필드의 모든 유저(관전자 포함) 화면 상단에 스폰 알림 배너가 즉시 표시된다.
3. 배너는 해당 블록의 이름과 보상 크레딧 정보를 포함한다.
4. 배너는 2.5초 후 자동으로 사라진다.

---

### [STORY 1-5] 유저는 블록 파괴 시 보상 알림 배너를 볼 수 있어야 한다.
1. 플레이어의 곡괭이가 Gold, Netherite, Diamond, 또는 Jackpot 블록을 파괴한다.
2. 같은 필드의 모든 유저 화면에 파괴 알림 배너가 표시된다.
3. **Gold / Netherite / Diamond 블록 파괴**: 배너에 "{닉네임} mined {블록명}! +{보상}" 문구가 3초 동안 표시된다.
4. **Jackpot 블록 파괴**: 특별 스타일 배너에 "{닉네임} mined Jackpot Block! +{보상}" 문구가 5초 동안 표시된다.

---

### [STORY 1-6] 유저는 곡괭이 5종의 비주얼을 구분할 수 있어야 한다.
1. 게임 화면에 등장하는 곡괭이는 8비트 픽셀 아트 스타일로 렌더링된다.
2. 곡괭이 종류별 색상:
   - **Basic**: 갈색
   - **Power**: 청록색
   - **Light**: 금색
   - **Swift**: 은색
   - **Elite**: 보라색
3. 유저는 색상으로 곡괭이 종류를 즉시 구분할 수 있다.

---

### [STORY 1-7] 유저는 알림 토글로 스폰/보상 알림을 독립적으로 켜고 끌 수 있어야 한다.
1. 화면 우상단에 알림 토글 버튼이 표시된다.
2. 두 가지 알림 채널을 독립적으로 on/off할 수 있다:
   - **Spawn 알림**: 희귀 블록 스폰 시 배너
   - **Reward 알림**: 블록 파괴 보상 배너
3. 토글 상태는 즉시 반영되며, 꺼진 채널의 배너는 표시되지 않는다.

---

### [STORY 1-8] 모든 알림은 8-bit 픽셀 스타일이어야 한다.
1. 스폰 알림, 보상 알림 등 모든 배너는 8-bit 레트로 스타일로 표시된다.
2. 모노스페이스 폰트, 2px 보더, 반투명 배경을 적용한다.
3. 다수의 배너가 동시 발생하면 우선순위(Jackpot > Diamond > Netherite > Gold)에 따라 순서대로 표시된다.

---

### [STORY 1-9] 유저는 모바일에서도 게임을 정상적으로 관전할 수 있어야 한다.
1. 스마트폰 또는 태블릿 브라우저로 접속하면 화면 크기에 맞춰 자동 조정된다.
2. 상단 HUD, 하단 버튼, 팝업 및 패널이 터치에 적합하게 표시된다.
3. 화면 확대/축소는 자동으로 방지된다.

---

### [STORY 1-10] 모바일에서 가로 모드 전환 시 세로 모드 안내가 표시되어야 한다.
1. 모바일 기기에서 가로 모드(landscape)로 접속하거나 전환하면 "세로 모드를 사용해 주세요" 안내 오버레이가 표시된다.
2. 오버레이는 세로 모드로 돌아올 때까지 유지된다.
3. 게임은 오버레이 뒤에서 계속 실행된다.

---

### [STORY 1-11] 서버 연결이 끊어지면 자동 재연결을 시도해야 한다.
1. 인터넷 연결이 끊어지면 게임 화면이 마지막 수신 상태에서 정지된다.
2. 화면에 "연결 끊김 - 재연결 중..." 오버레이가 표시된다.
3. 서버와의 연결이 자동으로 재시도된다.
4. 재연결 성공 시 서버로부터 전체 게임 상태 스냅샷을 수신하고, 화면이 최신 상태로 갱신된다.
5. 재연결 실패(3회 이상 시도 후) 시 "서버와의 연결이 끊겼습니다. 새로고침 해주세요." 메시지를 표시한다.

---

### [STORY 1-12] 최초 접속 시 서버 미응답이면 오류 화면을 표시해야 한다.
1. 최초 접속 시 서버가 응답하지 않으면 "서버에 연결할 수 없습니다. 잠시 후 다시 시도해 주세요." 오류 메시지를 표시한다.
2. 자동으로 5초마다 재시도하며, 카운트다운으로 안내한다.

---

### [STORY 1-13] 비로그인 유저는 채팅을 읽기만 할 수 있어야 한다.
1. 게임 화면 하단에 채팅 영역이 항상 표시된다.
2. 현재 필드의 플레이어 채팅 메시지가 실시간으로 나타난다.
3. 비로그인 상태에서는 채팅 메시지를 읽을 수만 있고, 전송은 불가하다.
4. 채팅 전송을 시도하면 로그인 안내 화면이 표시된다.
