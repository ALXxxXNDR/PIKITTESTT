## ✦ [EPIC 4] 필드 선택 서비스

### 목적
PIKIT은 두 가지 게임 필드를 제공한다: **일반 모드(Normal, 1배)** — 기본 비용과 보상, **하드코어 모드(Hardcore, 10배)** — 모든 비용과 보상이 10배. 유저는 언제든지 필드를 전환할 수 있다.

---

### [STORY 4-1] 유저는 Normal 필드에서 Hardcore 필드로 처음 전환 시 경고 팝업을 보아야 한다.

1. 유저가 Normal 필드에서 게임을 하고 있거나 관전 중이다.
2. 화면 상단의 필드 전환 스위치를 Hardcore(10배) 방향으로 클릭한다.
3. 브라우저 저장소에 "다시 묻지 않기" 설정이 저장되어 있는지 확인한다. 저장되어 있지 않은 경우 경고 팝업을 표시한다.
4. 스위치는 클릭 즉시 원래 상태(Normal)로 되돌아간다. 유저가 확인하기 전까지 실제 전환이 일어나지 않는다.
5. 화면 중앙에 경고 팝업이 오버레이로 나타난다.
6. 팝업 구성 요소:
   - 🔥 아이콘
   - 제목: "Hardcore Mode"
   - 설명: 하드코어 10배 모드에 진입한다는 안내. 모든 비용과 보상이 10배로 적용되므로 주의하라는 내용.
   - "다시 묻지 않기" 체크박스 (기본값: 미체크)
   - "취소(Cancel)" 버튼
   - "확인(OK)" 버튼

---

### [STORY 4-2] 유저가 Hardcore 전환 팝업에서 확인을 클릭하면 Hardcore 필드로 전환되어야 한다.

1. 경고 팝업이 표시된 상태에서 유저가 "확인(OK)" 버튼을 클릭한다.
2. "다시 묻지 않기" 체크박스가 체크된 경우, 이 설정이 브라우저 저장소에 저장되어 이후 경고 팝업이 표시되지 않는다.
3. 팝업이 닫힌다.
4. 필드 전환 스위치가 Hardcore 상태로 바뀐다.
5. 상점의 모든 가격이 10배로 갱신된다.
   - Basic 곡괭이: 3,400 → 34,000 크레딧
   - Power 곡괭이: 8,800 → 88,000 크레딧
   - Light 곡괭이: 3,900 → 39,000 크레딧
   - Swift 곡괭이: 3,600 → 36,000 크레딧
   - TNT: 8,000 → 80,000 크레딧
6. 게임 화면이 Hardcore 필드의 실시간 상태로 전환된다.

---

### [STORY 4-3] 유저가 Hardcore 전환 팝업에서 취소를 클릭하면 Normal 필드를 유지해야 한다.

1. 경고 팝업이 표시된 상태에서 유저가 "취소(Cancel)" 버튼을 클릭한다.
2. 팝업이 닫힌다.
3. 필드 전환 스위치가 원래 상태(Normal)로 그대로 유지된다. (팝업 표시 전에 이미 되돌아간 상태)
4. 서버로 필드 전환 요청이 전송되지 않는다.
5. 유저는 계속 Normal 필드에 머물며, 상점 가격도 1배 그대로 유지된다.

---

### [STORY 4-4] '다시 묻지 않기'를 체크하고 확인을 클릭한 후 두 번째 Hardcore 전환 시도 시 팝업이 표시되지 않아야 한다.

1. 유저가 경고 팝업에서 "다시 묻지 않기"를 체크하고 "확인(OK)"을 클릭한다.
2. 브라우저 저장소에 경고 건너뛰기 설정이 저장된다.
3. 유저가 Normal 필드로 다시 전환한 뒤, 이후 다시 Hardcore 전환을 시도한다.
4. 브라우저 저장소에서 설정을 확인하고 경고 팝업을 건너뛴다.
5. 즉시 Hardcore 필드로 전환된다. 팝업 없이 바로 가격이 10배로 바뀌고 게임 화면이 전환된다.

---

### [STORY 4-5] '다시 묻지 않기' 저장 후 브라우저를 닫고 다시 접속해도 팝업이 나타나지 않아야 한다.

1. "다시 묻지 않기" 설정이 브라우저 저장소에 저장된 상태에서 유저가 브라우저를 닫는다.
2. 브라우저 저장소(localStorage)는 창을 닫아도 유지된다.
3. 다시 접속하여 Hardcore 전환을 시도하면 경고 팝업 없이 즉시 전환된다.

---

### [STORY 4-6] 유저가 Hardcore 필드에서 Normal 필드로 전환할 수 있어야 한다.

1. 유저가 Hardcore 필드에서 게임 중이다.
2. 필드 전환 스위치를 클릭해 Normal 방향으로 전환한다.
3. Hardcore에서 Normal로의 전환에는 경고 팝업이 표시되지 않는다. 즉시 전환이 이루어진다.
4. 필드 전환 스위치가 Normal 상태로 바뀐다.
5. 상점 가격이 1배 기준으로 복귀한다.
   - Basic 곡괭이: 3,400 크레딧
   - Power 곡괭이: 8,800 크레딧
   - Light 곡괭이: 3,900 크레딧
   - Swift 곡괭이: 3,600 크레딧
   - TNT: 8,000 크레딧
6. 게임 화면이 Normal 필드의 실시간 상태로 전환된다.

---

### [STORY 4-7] 유저는 필드 전환 시 이전 필드에서 낙하 중이던 곡괭이가 사라진다는 것을 알아야 한다.

1. 유저가 Normal 필드에서 곡괭이를 구매해 낙하 중인 상태다.
2. 유저가 Hardcore 필드로 전환한다.
3. 이전 필드(Normal)에서 해당 유저의 플레이어 정보가 제거되며, 해당 유저 소유의 활성 곡괭이도 함께 사라진다.
4. 유저는 Normal 필드에 더 이상 활성 곡괭이가 없으며, 해당 곡괭이로부터 추가 보상을 받지 못한다.
5. Hardcore 필드에서 채굴을 재개하려면 새로 곡괭이를 구매해야 한다.
6. 주의: 현재 필드 전환 시 곡괭이 소멸에 대한 별도 경고는 표시되지 않는다.

---

### [STORY 4-8] 로그인하지 않은 상태에서 필드 전환을 시도하면 관전 화면만 해당 필드로 전환되어야 한다.

1. 로그인하지 않은 관전자가 필드 전환 스위치를 클릭해 Hardcore로 전환을 시도한다.
2. 경고 팝업이 동일하게 표시된다. 로그인 여부와 관계없이 팝업 절차는 동일하다.
3. "확인(OK)"을 클릭하면 Hardcore 필드의 관전 화면으로 전환된다.
4. 관전자는 Hardcore 필드의 실시간 게임 상황을 구경할 수 있다.
5. 곡괭이 또는 TNT 구매를 시도하면 로그인 안내가 표시된다.

### 정책 요약

| 항목 | 내용 |
|------|------|
| 기본 필드 | Normal (1배) |
| Normal → Hardcore 전환 | 첫 전환 시 경고 팝업 표시 |
| Hardcore → Normal 전환 | 경고 없이 즉시 처리 |
| "다시 묻지 않기" 저장 위치 | 브라우저 저장소 (창 닫아도 유지) |
| 필드 전환 시 곡괭이 처리 | 이전 필드의 활성 곡괭이 즉시 제거 |
