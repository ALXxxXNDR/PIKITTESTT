# [Epic 4] 필드 선택 서비스
> 작성 기준: v5.0 | 작성일: 2026-03-24
> 관점: 유저 플로우 (비개발자용)

## 목적
유저가 Normal(1x)과 Hardcore(10x) 두 가지 필드를 자유롭게 전환하며 게임에 참여하거나 관전할 수 있도록 한다.

---

### [STORY 4-1] 유저는 Normal 필드에서 Hardcore 필드로 처음 전환 시 경고 팝업을 보아야 한다.
1. 유저가 화면 상단의 필드 전환 스위치를 Hardcore(10x) 방향으로 클릭한다.
2. 스위치는 클릭 즉시 원래 상태(Normal)로 되돌아간다. 유저가 확인하기 전까지 실제 전환이 일어나지 않는다.
3. 화면 중앙에 경고 팝업이 오버레이로 나타난다.
4. 팝업 구성:
   - 🔥 아이콘
   - 제목: "Hardcore Mode"
   - 설명: 모든 비용과 보상이 10배로 적용된다는 경고
   - "다시 묻지 않기" 체크박스 (기본값: 미체크)
   - "취소(Cancel)" / "확인(OK)" 버튼

---

### [STORY 4-2] 유저가 경고 팝업에서 확인을 클릭하면 Hardcore 필드로 전환되어야 한다.
1. 유저가 "확인(OK)" 버튼을 클릭한다.
2. "다시 묻지 않기" 체크 시 localStorage에 영구 저장되어 이후 팝업이 표시되지 않는다.
3. 팝업이 닫히고 필드 전환 스위치가 Hardcore 상태로 바뀐다.
4. 상점의 모든 가격이 10배로 갱신된다.
5. 게임 화면이 Hardcore 필드의 실시간 상태로 전환된다.

---

### [STORY 4-3] 유저가 경고 팝업에서 취소를 클릭하면 Normal 필드를 유지해야 한다.
1. 유저가 "취소(Cancel)" 버튼을 클릭한다.
2. 팝업이 닫히고 Normal 필드가 그대로 유지된다.
3. 서버로 전환 요청이 전송되지 않는다.

---

### [STORY 4-4] "다시 묻지 않기" 설정은 브라우저를 닫아도 유지되어야 한다.
1. "다시 묻지 않기"를 체크하고 확인을 클릭하면 localStorage에 저장된다.
2. 이후 Hardcore 전환 시도 시 팝업 없이 즉시 전환된다.
3. 브라우저를 닫고 다시 접속해도 설정이 유지된다.

---

### [STORY 4-5] Hardcore에서 Normal로의 전환은 즉시 이루어져야 한다.
1. Hardcore 필드에서 필드 전환 스위치를 Normal 방향으로 클릭한다.
2. 경고 팝업 없이 즉시 Normal 필드로 전환된다.
3. 상점 가격이 1배 기준으로 복귀한다.

---

### [STORY 4-6] 필드 전환 시 활성 곡괭이는 즉시 소멸되어야 한다.
1. 유저가 활성 곡괭이가 낙하 중인 상태에서 필드를 전환한다.
2. 이전 필드의 모든 활성 곡괭이와 낙하 중인 TNT는 즉시 소멸한다.
3. 구매 크레딧은 환불되지 않는다.
4. 새 필드에서 채굴을 재개하려면 곡괭이를 새로 구매해야 한다.

---

### [STORY 4-7] 비로그인 유저도 필드를 전환해 관전할 수 있어야 한다.
1. 비로그인 관전자가 필드 전환 스위치를 클릭한다.
2. 로그인 유저와 동일한 팝업 절차를 거친다.
3. 확인 시 해당 필드의 관전 화면으로 전환된다.
4. 곡괭이/TNT 구매를 시도하면 로그인 안내가 표시된다.

---

### [STORY 4-8] 필드 전환 시 채팅 내역이 새 필드의 내역으로 교체되어야 한다.
1. 필드를 전환하면 이전 필드의 채팅 내역이 지워진다.
2. 새 필드의 최근 채팅 내역으로 교체된다.
3. 필드 전환 시스템 메시지가 채팅창에 표시된다.

---

## 정책 요약

| 항목 | 내용 |
|------|------|
| 필드 종류 | Normal (1x), Hardcore (10x) |
| 기본 필드 | Normal |
| Normal → Hardcore | 첫 전환 시 경고 팝업 |
| Hardcore → Normal | 즉시 전환 |
| "다시 묻지 않기" | localStorage 영구 저장 |
| 필드 전환 시 곡괭이 | 즉시 소멸, 환불 없음 |
| 비로그인 관전 전환 | 가능 |
