## ✦ [EPIC 7] 블록 보상 및 알림 서비스

### 목적
게임 필드에는 10종의 블록이 다양한 확률로 등장한다. 각 블록마다 체력(HP)과 보상이 다르며, 희귀 블록이 등장하거나 파괴될 때 모든 유저에게 알림이 표시된다.

### 블록 보상 정보 (v4.7 기준)

| 블록 이름 | 등장 확률 | 체력 | 보상 |
|----------|---------|------|------|
| Jackpot Block | 특수 조건 | 1,500 | 동적 풀 전액 (현재 풀 금액) |
| Diamond Block | 1% | 180 | 4,500 크레딧 |
| Gold Block | 2% | 90 | 1,800 크레딧 |
| Emerald Block | 5% | 55 | 540 크레딧 |
| Iron Block | 12% | 20 | 100 크레딧 |
| Copper Block | 20% | 15 | 50 크레딧 |
| Stone | 20% | 3 | 28 크레딧 |
| Dirt | 18% | 2 | 22 크레딧 |
| Gravel | 12% | 3 | 25 크레딧 |
| Clay | 10% | 2 | 24 크레딧 |

---

### [STORY 7-1] 유저는 메뉴 버튼을 클릭해 블록 가이드를 확인할 수 있어야 한다.

1. 유저가 화면 하단의 **[MENU]** 버튼을 클릭한다.
2. 열려 있던 다른 패널(예: 숍)이 있으면 자동으로 먼저 닫힌다.
3. 메뉴 패널이 화면 측면에서 슬라이드 인 되며 열린다.
4. 패널 상단에는 **게임 정보** 섹션이 표시된다:
   - 현재 필드에 접속 중인 플레이어 수
   - 현재 필드에서 활동 중인 곡괭이 수
5. 패널 중단에는 **블록 가이드** 섹션이 표시된다:
   - 10종 블록 전체 목록 (Bedrock 벽 제외)
   - 각 항목: 블록 아이콘, 블록 이름, 보상 크레딧, 등장 확률
   - 보상이 높은 순서대로 정렬되어 표시됨

---

### [STORY 7-2] 유저는 메뉴 패널을 닫을 수 있어야 한다.

**방법 A — X 버튼으로 닫기:**

1. 유저가 메뉴 패널 상단의 **[×]** 버튼을 클릭한다.
2. 메뉴 패널이 슬라이드 아웃 되며 닫힌다.
3. 다른 패널(숍 등)이 열려 있지 않으면 반투명 배경도 함께 사라진다.

**방법 B — 외부 어두운 영역 클릭으로 닫기:**

1. 메뉴 패널이 열린 상태에서 패널 바깥쪽 어두운 영역을 클릭한다.
2. 메뉴 패널을 포함한 모든 열린 패널이 한 번에 닫힌다.
3. 반투명 배경도 함께 사라진다.

---

### [STORY 7-3] 유저의 곡괭이가 일반 블록(돌/흙/자갈/점토)을 파괴하면 즉시 보상이 지급되어야 한다.

1. 유저의 곡괭이가 낙하하며 일반 블록(돌·흙·자갈·점토 중 하나)에 충돌한다.
2. 해당 블록은 곡괭이 1회 충돌만으로 즉시 파괴된다.
   - 돌(Stone): 28 크레딧 지급
   - 흙(Dirt): 22 크레딧 지급
   - 자갈(Gravel): 25 크레딧 지급
   - 점토(Clay): 24 크레딧 지급
3. 파괴된 블록은 다음 화면 갱신에서 필드에서 사라진다.
4. 화면 상단의 유저 잔액이 해당 보상만큼 즉시 증가한다.

---

### [STORY 7-4] 유저의 곡괭이가 철(Iron Block)을 공격하면 여러 번 타격 후 파괴되고 보상이 지급되어야 한다.

1. 유저의 곡괭이가 철 블록(Iron Block)에 충돌한다.
2. 철 블록은 체력(HP)이 20이므로 한 번에 파괴되지 않는다.
   - 기본 곡괭이(공격력 3): 7번 충돌 필요
   - 라이트 곡괭이(공격력 4): 5번 충돌 필요
   - 파워 곡괭이(공격력 5): 4번 충돌 필요
3. 곡괭이가 충돌할 때마다 블록 표면에 균열이 점진적으로 진해진다 (총 10단계).
4. 마지막 타격으로 체력이 0 이하가 되면 블록이 파괴되고 **100 크레딧**이 지급된다.
5. 중간 타격에는 보상이 없으며, 최종 파괴 순간에만 보상이 지급된다.
6. 화면 상단의 유저 잔액이 즉시 증가한다.

---

### [STORY 7-5] 유저의 곡괭이가 구리(Copper Block)를 파괴하면 50 크레딧이 지급되어야 한다.

1. 유저의 곡괭이가 구리 블록(Copper Block)을 여러 번 타격하여 체력을 모두 소진시킨다.
   - 체력 15이므로 기본 곡괭이 기준 5번 충돌 필요
2. 최종 파괴 시 **50 크레딧**이 지급되며 화면 상단 잔액이 즉시 증가한다.
3. TNT를 사용하면 한 번의 폭발로도 즉시 파괴 가능하다.

---

### [STORY 7-6] 유저의 곡괭이가 에메랄드(Emerald Block)를 파괴하면 540 크레딧이 지급되어야 한다.

1. 유저의 곡괭이가 에메랄드 블록(Emerald Block)을 반복 타격하여 체력을 모두 소진시킨다.
   - 체력 55이므로 기본 곡괭이 기준 약 19번 충돌 필요
2. 최종 파괴 시 **540 크레딧**이 지급되며 화면 상단 잔액이 즉시 증가한다.
3. 보상이 1,000 크레딧 미만이므로 오른쪽 잭팟 피드에는 표시되지 않는다.

---

### [STORY 7-7] 유저의 곡괭이가 Diamond Block을 파괴하면 4,500 크레딧이 지급되고 전체 알림 배너가 표시되어야 한다.

1. 유저의 곡괭이가 Diamond Block을 반복 타격하여 체력 180을 모두 소진시킨다.
2. 최종 파괴 순간 **4,500 크레딧**이 해당 유저의 잔액에 즉시 적립된다.
3. 현재 필드의 모든 유저 화면에 전체 알림 배너가 표시된다:
   - 배너 내용: "[파괴한 플레이어 이름] mined Diamond Block! +4,500"
   - 다이아몬드 색상의 배너 스타일로 표시됨
   - 파괴자 본인 화면에도 동일하게 표시됨
4. 배너는 **10초** 후 자동으로 서서히 사라진다.
5. 4,500 크레딧이 1,000 크레딧 이상이므로 화면 오른쪽 잭팟 피드에도 항목이 추가된다.

---

### [STORY 7-8] Diamond Block이 새로 등장하면 전체 유저에게 스폰 알림 배너가 표시되어야 한다.

1. 새 구역이 생성될 때 Diamond Block이 해당 구역에 배치된다 (1% 확률).
2. 현재 필드의 모든 유저 화면에 스폰 알림 배너가 표시된다:
   - 배너 내용: "💎 Diamond Block이 등장했습니다! (4,500 크레딧) 💎"
3. 배너는 **4초** 후 자동으로 서서히 사라진다.
4. 유저는 이 알림을 보고 Diamond Block을 향해 곡괭이를 집중 투입하는 전략을 세울 수 있다.
5. 게임이 막 시작된 직후 초기 구역 생성 시에는 이 알림이 표시되지 않는다.

---

### [STORY 7-9] Gold Block이 파괴되면 전체 유저에게 파괴 알림 배너가 표시되어야 한다.

1. 새 구역이 생성될 때 Gold Block이 해당 구역에 배치된다 (2% 확률).
2. Gold Block이 새로 등장할 때는 별도의 스폰 알림 배너가 표시되지 않는다. (Diamond Block, Jackpot Block만 스폰 알림 대상)
3. Gold Block이 파괴되면 현재 필드의 모든 유저 화면에 파괴 알림 배너가 **10초**간 표시된다.
   - 배너 내용: "[파괴자 이름] mined Gold Block! +[보상 크레딧]"
   - 골드 색상 배너, ✨ 이모지 포함

---

### [STORY 7-10] 스폰 알림 배너는 4초 후 자동으로 서서히 사라져야 한다.

1. Diamond Block 또는 Jackpot Block 등장 알림 배너가 화면에 표시된다. (Gold Block은 스폰 알림 없음)
2. 배너가 나타난 지 **4초**가 경과하면 배너가 서서히 투명해지기 시작한다.
3. 약 0.5초에 걸쳐 완전히 투명해진 후 화면에서 제거된다.
4. 여러 알림이 동시에 발생한 경우 각 배너는 서로 독립적으로 카운트되어 각자 4초 후 사라진다.
   - 예: Diamond Block 알림이 표시된 2초 후 Jackpot Block 알림이 추가로 표시되면, Diamond 알림은 2초 후, Jackpot 알림은 4초 후 사라짐

---

### [STORY 7-11] Diamond Block 또는 Gold Block이 파괴되면 파괴자 이름을 포함한 전체 배너 알림이 10초간 표시되어야 한다.

1. 어떤 유저의 곡괭이가 Diamond Block 또는 Gold Block을 최종 파괴한다.
2. 현재 필드의 모든 유저 화면에 파괴 알림 배너가 표시된다.
   - Diamond Block 파괴 시: 다이아몬드 색상 배너, 💎 이모지 포함
   - Gold Block 파괴 시: 골드 색상 배너, ✨ 이모지 포함
   - 배너 내용: "[파괴자 이름] mined [블록 이름]! +[보상 크레딧]"
3. 파괴자 본인 화면에도 동일 배너가 표시된다.
4. 배너는 **10초** 후 자동으로 사라진다.

---

### [STORY 7-12] 유저는 화면 측면의 피드에서 고가 보상 이벤트를 실시간으로 확인할 수 있어야 한다.

1. 필드에서 1,000 크레딧 이상의 보상이 발생하면 화면 측면 피드에 새 항목이 추가된다.
   - Diamond Block 파괴: 4,500 크레딧 → 피드 추가
   - Gold Block 파괴: 1,800 크레딧 → 피드 추가
   - Jackpot Block 파괴: 동적 풀 전액 → 피드 추가
   - TNT 폭발 총 보상이 5,000 크레딧 이상인 경우 → 피드 추가
2. 각 항목에는 플레이어 이름, 획득한 블록 이름, 보상 크레딧이 표시된다.
   - 표시 형태: "[플레이어명] found [보상]cr from [블록명]!"
3. 항목이 추가된 지 **4초** 후 자동으로 피드에서 사라진다.
4. 최근 5개의 항목만 피드에 표시된다.

---

### [STORY 7-13] 잭팟 블록이 등장하면 전체 유저에게 특별 알림 배너가 표시되어야 한다.

1. 잭팟 등장 조건이 충족되고 잭팟 블록이 필드에 스폰된다 (등장 조건은 [Epic 8] 참조).
2. 현재 필드의 모든 유저 화면에 특별 배너가 표시된다:
   - 배너 내용: "🏆 Jackpot Block이 등장했습니다! (현재 풀 금액) 🏆"
   - 황금색 특별 스타일의 배너로 표시됨
3. 배너는 **4초** 후 자동으로 서서히 사라진다.
4. 유저들은 이 알림을 보고 잭팟 블록을 집중 공략하여 대형 보상을 노릴 수 있다.
