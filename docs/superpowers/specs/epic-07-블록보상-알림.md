## ✦ [EPIC 7] 블록 보상 및 알림 서비스

### 목적
게임 필드에는 10종 + Jackpot 블록이 다양한 확률로 등장한다. 각 블록마다 체력(HP)과 보상이 다르며, 희귀 블록이 등장하거나 파괴될 때 모든 유저에게 알림이 표시된다. 환수율 90% 설계(하우스 10%)에 의해 블록 보상과 등장 확률이 수학적으로 조정되어 있다.

### 블록 보상 정보 (v5.0 기준)

| 블록 이름 | 등장 확률 | 체력 | 보상 |
|----------|---------|------|------|
| Jackpot Block | 특수 조건 | 1,500 | 동적 풀 전액 |
| Diamond Block | 3.85% | 2,000 | 40,000 크레딧 |
| Netherite Block | 4.62% | 1,110 | 15,916 크레딧 |
| Gold Block | 5.55% | 616 | 6,333 크레딧 |
| Emerald Block | 6.66% | 342 | 2,520 크레딧 |
| Iron Block | 7.99% | 190 | 1,003 크레딧 |
| Copper Block | 9.59% | 105 | 399 크레딧 |
| Stone | 11.50% | 58 | 159 크레딧 |
| Gravel | 13.80% | 32 | 63 크레딧 |
| Clay | 16.56% | 18 | 25 크레딧 |
| Dirt | 19.88% | 10 | 10 크레딧 |

### [STORY 7-1] 유저는 메뉴 버튼을 클릭해 블록 가이드를 확인할 수 있어야 한다.

1. 유저가 화면 하단의 **[MENU]** 버튼을 클릭한다.
2. 열려 있던 다른 패널(예: 숍)이 있으면 자동으로 먼저 닫힌다.
3. 메뉴 패널이 화면 측면에서 슬라이드 인 되며 열린다.
4. 패널 상단에는 **게임 정보** 섹션이 표시된다:
   - 현재 필드에 접속 중인 플레이어 수
   - 현재 필드에서 활동 중인 곡괭이 수
5. 패널 중단에는 **블록 가이드** 섹션이 표시된다:
   - 11종 블록 전체 목록 (Bedrock 벽 제외, Netherite 포함)
   - 각 항목: 블록 아이콘, 블록 이름, 보상 크레딧, 등장 확률
   - 보상이 높은 순서대로 정렬되어 표시됨

---

### [STORY 7-2] 유저는 메뉴 패널을 닫을 수 있어야 한다.

**방법 A — X 버튼으로 닫기:**

1. 유저가 메뉴 패널 상단의 **[×]** 버튼을 클릭한다.
2. 메뉴 패널이 슬라이드 아웃 되며 닫힌다.
3. 다른 패널(숍 등)이 열려 있지 않으면 반투명 배경도 함께 사라진다.

**방법 B — 외부 어두운 영역 클릭으로 닫기:**

1. 메뉴 패널이 열린 상태에서 패널 바깥쪽 어두운 영역을 클릭한다.
2. 메뉴 패널을 포함한 모든 열린 패널이 한 번에 닫힌다.
3. 반투명 배경도 함께 사라진다.

---

### [STORY 7-3] 유저의 곡괭이가 일반 블록(흙/점토/자갈/돌)을 파괴하면 즉시 보상이 지급되어야 한다.

1. 유저의 곡괭이가 낙하하며 일반 블록(흙·점토·자갈·돌 중 하나)에 충돌한다.
2. 이 블록들은 체력이 높아져 기본 곡괭이(DMG 14) 1회 충돌로는 파괴되지 않는다.
   - 흙(Dirt): HP 10 → 1번 충돌로 파괴, **10 크레딧** 지급
   - 점토(Clay): HP 18 → 2번 충돌 필요, **25 크레딧** 지급
   - 자갈(Gravel): HP 32 → 3번 충돌 필요, **63 크레딧** 지급
   - 돌(Stone): HP 58 → 5번 충돌 필요, **159 크레딧** 지급
3. 파괴된 블록은 다음 화면 갱신에서 필드에서 사라진다.
4. 화면 상단의 유저 잔액이 해당 보상만큼 즉시 증가한다.

---

### [STORY 7-4] 유저의 곡괭이가 철(Iron Block)을 공격하면 여러 번 타격 후 파괴되고 보상이 지급되어야 한다.

1. 유저의 곡괭이가 철 블록(Iron Block)에 충돌한다.
2. 철 블록은 체력(HP)이 190이므로 한 번에 파괴되지 않는다.
   - 기본 곡괭이(공격력 14): 약 14번 충돌 필요
3. 곡괭이가 충돌할 때마다 블록 표면에 균열이 점진적으로 진해진다 (총 10단계).
4. 마지막 타격으로 체력이 0 이하가 되면 블록이 파괴되고 **1,003 크레딧**이 지급된다.
5. 중간 타격에는 보상이 없으며, 최종 파괴 순간에만 보상이 지급된다.
6. 화면 상단의 유저 잔액이 즉시 증가한다.

---

### [STORY 7-5] 유저의 곡괭이가 구리(Copper Block)를 파괴하면 399 크레딧이 지급되어야 한다.

1. 유저의 곡괭이가 구리 블록(Copper Block)을 여러 번 타격하여 체력을 모두 소진시킨다.
   - 체력 105이므로 기본 곡괭이(DMG 14) 기준 약 8번 충돌 필요
2. 최종 파괴 시 **399 크레딧**이 지급되며 화면 상단 잔액이 즉시 증가한다.
3. TNT를 사용하면 한 번의 폭발로도 즉시 파괴 가능하다.

---

### [STORY 7-6] 유저의 곡괭이가 에메랄드(Emerald Block)를 파괴하면 2,520 크레딧이 지급되어야 한다.

1. 유저의 곡괭이가 에메랄드 블록(Emerald Block)을 반복 타격하여 체력을 모두 소진시킨다.
   - 체력 342이므로 기본 곡괭이(DMG 14) 기준 약 25번 충돌 필요
2. 최종 파괴 시 **2,520 크레딧**이 지급되며 화면 상단 잔액이 즉시 증가한다.
3. 보상이 2,000 크레딧 이상이므로 화면 오른쪽 고가 보상 피드에 표시된다.

---

### [STORY 7-7] 유저의 곡괭이가 Diamond Block을 파괴하면 40,000 크레딧이 지급되고 전체 알림 배너가 표시되어야 한다.

1. 유저의 곡괭이가 Diamond Block을 반복 타격하여 체력 2,000을 모두 소진시킨다.
2. 최종 파괴 순간 **40,000 크레딧**이 해당 유저의 잔액에 즉시 적립된다.
3. 현재 필드의 모든 유저 화면에 전체 알림 배너가 표시된다:
   - 배너 내용: "[파괴한 플레이어 이름] mined Diamond Block! +40,000"
   - 다이아몬드 색상의 배너 스타일로 표시됨
   - 파괴자 본인 화면에도 동일하게 표시됨
4. 배너는 **3초** 후 자동으로 서서히 사라진다.
5. 40,000 크레딧이 2,000 크레딧 이상이므로 화면 오른쪽 고가 보상 피드에도 항목이 추가된다.

---

### [STORY 7-8] Diamond Block이 새로 등장하면 전체 유저에게 스폰 알림 배너가 표시되어야 한다.

1. 새 구역이 생성될 때 Diamond Block이 해당 구역에 배치된다 (3.85% 확률).
2. 현재 필드의 모든 유저 화면에 스폰 알림 배너가 표시된다:
   - 배너 내용: "💎 Diamond Block이 등장했습니다! (40,000 크레딧) 💎"
3. 배너는 **2.5초** 후 자동으로 서서히 사라진다.
4. 유저는 이 알림을 보고 Diamond Block을 향해 곡괭이를 집중 투입하는 전략을 세울 수 있다.
5. 게임이 막 시작된 직후 초기 구역 생성 시에는 이 알림이 표시되지 않는다.

---

### [STORY 7-9] Gold Block이 파괴되면 전체 유저에게 파괴 알림 배너가 표시되어야 한다.

1. 새 구역이 생성될 때 Gold Block이 해당 구역에 배치된다 (5.55% 확률).
2. Gold Block이 새로 등장할 때는 별도의 스폰 알림 배너가 표시되지 않는다. (Diamond Block, Netherite Block, Jackpot Block만 스폰 알림 대상)
3. Gold Block이 파괴되면 현재 필드의 모든 유저 화면에 파괴 알림 배너가 **3초**간 표시된다.
   - 배너 내용: "[파괴자 이름] mined Gold Block! +6,333"
   - 골드 색상 배너, ✨ 이모지 포함

---

### [STORY 7-10] 스폰 알림 배너는 2.5초 후 자동으로 서서히 사라져야 한다.

1. Diamond Block, Netherite Block 또는 Jackpot Block 등장 알림 배너가 화면에 표시된다. (Gold Block은 스폰 알림 없음)
2. 배너가 나타난 지 **2.5초**가 경과하면 배너가 서서히 투명해지기 시작한다.
3. 약 0.5초에 걸쳐 완전히 투명해진 후 화면에서 제거된다.
4. 여러 알림이 동시에 발생한 경우 각 배너는 서로 독립적으로 카운트되어 각자 2.5초 후 사라진다.
   - 예: Diamond Block 알림이 표시된 1초 후 Jackpot Block 알림이 추가로 표시되면, Diamond 알림은 1.5초 후, Jackpot 알림은 2.5초 후 사라짐

---

### [STORY 7-11] Diamond Block, Gold Block 또는 Netherite Block이 파괴되면 파괴자 이름을 포함한 전체 배너 알림이 표시되어야 한다.

1. 어떤 유저의 곡괭이가 Diamond Block, Gold Block 또는 Netherite Block을 최종 파괴한다.
2. 현재 필드의 모든 유저 화면에 파괴 알림 배너가 표시된다.
   - Diamond Block 파괴 시: 다이아몬드 색상 배너, 💎 이모지 포함
   - Gold Block 파괴 시: 골드 색상 배너, ✨ 이모지 포함
   - Netherite Block 파괴 시: 다크 브라운 + 골드 배너, ⚔️ 이모지 포함
   - 배너 내용: "[파괴자 이름] mined [블록 이름]! +[보상 크레딧]"
3. 파괴자 본인 화면에도 동일 배너가 표시된다.
4. 배너는 **3초** 후 자동으로 사라진다. (잭팟 블록 파괴 시에는 **5초**)

---

### [STORY 7-12] 유저는 화면 측면의 피드에서 고가 보상 이벤트를 실시간으로 확인할 수 있어야 한다.

1. 필드에서 2,000 크레딧 이상의 보상이 발생하면 화면 측면 피드에 새 항목이 추가된다.
   - Diamond Block 파괴: 40,000 크레딧 → 피드 추가
   - Netherite Block 파괴: 15,916 크레딧 → 피드 추가
   - Gold Block 파괴: 6,333 크레딧 → 피드 추가
   - Emerald Block 파괴: 2,520 크레딧 → 피드 추가
   - Jackpot Block 파괴: 동적 풀 전액 → 피드 추가
   - TNT 폭발 총 보상이 5,000 크레딧 이상인 경우 → 피드 추가
2. 각 항목에는 플레이어 이름, 획득한 블록 이름, 보상 크레딧이 표시된다.
   - 표시 형태: "[플레이어명] found [보상]cr from [블록명]!"
3. 항목이 추가된 지 **4초** 후 자동으로 피드에서 사라진다.
4. 최근 5개의 항목만 피드에 표시된다.

---

### [STORY 7-13] 잭팟 블록이 등장하면 전체 유저에게 특별 알림 배너가 표시되어야 한다.

1. 잭팟 등장 조건이 충족되고 잭팟 블록이 필드에 스폰된다 (등장 조건은 [Epic 8] 참조).
2. 현재 필드의 모든 유저 화면에 특별 배너가 표시된다:
   - 배너 내용: "🏆 Jackpot Block이 등장했습니다! (현재 풀 금액) 🏆"
   - 황금색 특별 스타일의 배너로 표시됨
3. 배너는 **2.5초** 후 자동으로 서서히 사라진다.
4. 유저들은 이 알림을 보고 잭팟 블록을 집중 공략하여 대형 보상을 노릴 수 있다.

---

### [STORY 7-14] 동일 틱에 여러 곡괭이가 같은 블록에 마지막 타격을 가해도 보상은 한 명에게만 지급되어야 한다.

1. 서버가 동일 틱에 여러 곡괭이의 충돌을 처리할 때, 해당 틱에서 가장 먼저 처리된 곡괭이의 소유자가 블록 파괴 보상 전액을 수령한다.
2. 보상이 중복 지급되어서는 안 된다.
3. 두 번째 이후 곡괭이는 해당 블록이 이미 파괴됐으므로 충돌이 발생하지 않은 것으로 처리되어 다음 블록으로 이동한다.

---

### [STORY 7-15] TNT 폭발로 파괴된 블록도 퀘스트 카운터에 집계된다.

1. TNT 폭발 범위 내 블록이 파괴될 때, 해당 TNT를 구매한 유저의 퀘스트 카운터(총 블록 파괴 수, 타입별 블록 파괴 수)가 파괴된 블록 수만큼 증가한다.
2. 단, "곡괭이 구매" 관련 퀘스트는 TNT로 카운트되지 않는다.

---

### [STORY 7-16] Gold Block은 스폰 알림 없이 등장하는 것이 의도된 설계이다.

1. Gold Block은 스폰 시 별도의 등장 알림 배너를 표시하지 않는다.
2. 유저는 Gold Block이 파괴되었을 때 결과 배너를 통해 해당 블록의 존재를 인식한다.
3. 이는 Diamond Block보다 높은 긴장감과 발견의 즐거움을 주기 위한 의도된 게임 설계이다.

---

### [STORY 7-17] 블록 가이드에서 잭팟 블록의 보상은 현재 풀 금액으로 표시된다.

1. 블록 가이드 팝업에서 잭팟 블록의 보상 칸은 고정 수치 대신 "현재 풀: {금액} 크레딧" 형태로 실시간 금액을 표시한다.
2. 잭팟 풀이 0이거나 등장 조건(50,000cr 미만)이 충족되지 않으면 "아직 적립 중..." 으로 표시한다.

---

### [STORY 7-18] Netherite Block이 등장하면 스폰 알림과 함께 파괴 시 15,916 크레딧이 지급되어야 한다. ⚔️

1. 새 구역이 생성될 때 Netherite Block이 해당 구역에 배치된다 (4.62% 확률).
2. Netherite Block이 등장하면 전체 유저에게 스폰 알림 배너가 표시된다:
   - 배너 내용: "⚔️ Netherite Block이 등장했습니다! (15,916 크레딧) ⚔️"
   - 다크 브라운 + 골드 스펙 스타일의 배너로 표시됨
3. 배너는 **2.5초** 후 자동으로 서서히 사라진다.
4. Netherite Block의 체력은 1,110이며, tntResist 속성이 있어 TNT에 대한 내성이 있다.
5. 유저의 곡괭이가 Netherite Block을 반복 타격하여 체력을 모두 소진시키면 **15,916 크레딧**이 지급된다.
6. 최종 파괴 시 현재 필드의 모든 유저 화면에 파괴 알림 배너가 표시된다:
   - 배너 내용: "[파괴자 이름] mined Netherite Block! +15,916"
   - 다크 브라운 + 골드 배너, ⚔️ 이모지 포함
7. 파괴 배너는 **3초** 후 자동으로 사라진다.
8. 15,916 크레딧이 2,000 크레딧 이상이므로 화면 오른쪽 고가 보상 피드에도 항목이 추가된다.
9. 블록 가이드에서 Netherite Block의 비주얼은 다크 브라운 바탕에 골드 스펙이 박힌 8-bit 픽셀아트로 표시된다.

### [STORY 7-19] 유저는 알림을 종류별로 끄거나 켤 수 있어야 한다.

1. 화면 우상단에 두 개의 토글 버튼이 표시된다:
   - 📢 **Spawn** — 블록 등장 알림 토글
   - 💰 **Reward** — 블록 파괴 보상 배너 토글
2. 각 버튼은 독립적으로 켜고 끌 수 있다.
   - Spawn 끄기: Diamond, Netherite, Jackpot 등장 알림이 비활성화됨
   - Reward 끄기: 블록 파괴 시 전체 배너 알림이 비활성화됨
3. 토글 상태는 켜짐(활성) / 꺼짐(비활성)으로 시각적으로 구분된다.
   - 활성 상태: 밝은 색, 아이콘 강조
   - 비활성 상태: 어두운 색, 아이콘 흐림
4. 기본값은 둘 다 켜짐(활성)이다.
5. 화면 오른쪽 고가 보상 피드는 토글과 무관하게 항상 표시된다.

### [STORY 7-20] 모든 알림은 8-bit 픽셀 스타일로 표시된다.

1. 스폰 알림 배너, 파괴 보상 배너, 고가 보상 피드 등 모든 알림 UI는 8-bit 레트로 픽셀 스타일로 표시된다.
2. 스타일 규격:
   - 폰트: 모노스페이스 (예: Press Start 2P, Courier New fallback)
   - 테두리: 2px 픽셀 보더 (직선, 라운드 없음)
   - 배경: 반투명 (rgba, 불투명도 80~90%)
   - 그림자/광택 효과 없음 — 플랫한 레트로 느낌
3. 기존의 gradient, border-radius, box-shadow 스타일은 사용하지 않는다.
4. 시네마틱 효과는 간소화: 화이트 플래시 없음, 충격파 크기 축소, 파티클 수 감소.

## 정책 요약

- **블록 종류**: 10종 + Jackpot (Netherite 신규 추가)
- **보상 범위**: 10 크레딧(Dirt) ~ 40,000 크레딧(Diamond) + 동적 풀(Jackpot)
- **환수율**: 90% (하우스 10% — 블록 확률/보상 수학적 설계)
- **스폰 알림 대상**: Diamond, Netherite, Jackpot (2.5초 후 소멸)
- **파괴 배너 대상**: Diamond, Netherite, Gold (3초 후 소멸, Jackpot은 5초)
- **고가 보상 피드**: 2,000 크레딧 이상 (Emerald, Gold, Netherite, Diamond, Jackpot, TNT 5,000+)
- **알림 토글**: 📢 Spawn / 💰 Reward — 각각 독립적으로 on/off
- **UI 스타일**: 8-bit 픽셀 (모노스페이스, 2px 보더, 반투명 bg, 시네마틱 간소화)
- **동시 타격 보호**: 동일 틱 다중 타격 시 최초 1인만 보상
- **TNT 퀘스트 연동**: TNT로 파괴한 블록도 퀘스트 카운터에 포함
