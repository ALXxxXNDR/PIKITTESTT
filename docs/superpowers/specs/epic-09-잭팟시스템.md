## ✦ [EPIC 9] 잭팟 시스템 서비스

### 목적
플랫폼 하우스 수익의 일부를 누적 적립하여 특수 블록(잭팟 블록)을 공략한 최후의 타격자에게 지급하는 대형 보상 이벤트 시스템이다. 잭팟 블록은 체력이 높아 여러 플레이어가 협력하여 레이드 형식으로 파괴해야 하며, 최후 타격자가 누적된 풀 전액을 단독 수령한다.

### 잭팟 풀 적립 방식
- 플레이어의 곡괭이가 수명을 다할 때마다 해당 곡괭이의 하우스 수익(구매가 - 획득 보상)이 집계됨
- 하우스 누적 수익이 **50,000 크레딧** 달성할 때마다 **2,500 크레딧**이 잭팟 풀에 자동 추가
- 잭팟 풀은 게임이 진행되는 동안 계속 누적되며, 잭팟 블록이 파괴될 때 지급 후 초기화

### 잭팟 블록 등장 조건
- 잭팟 풀이 **50,000 크레딧 이상** 누적된 상태에서만 등장 가능
- 위 조건 충족 시 청크 블록 위치당 0.01% 확률로 등장
- 필드에 잭팟 블록이 이미 존재하면 추가 등장 불가

---

### [STORY 9-1] 잭팟 풀이 50,000 크레딧 미만이면 잭팟 블록이 등장하지 않아야 한다.

1. 현재 잭팟 풀이 50,000 크레딧에 미달한 상태이다.
2. 새로운 구역이 생성될 때 잭팟 블록 등장 여부를 판단한다.
3. 풀 조건이 미달되어 잭팟 블록은 등장하지 않는다.
4. 해당 구역에는 일반 블록만 배치되어 게임이 정상적으로 진행된다.

---

### [STORY 9-2] 잭팟 풀이 50,000 크레딧 이상이면 잭팟 블록이 0.01% 확률로 등장할 수 있어야 한다.

1. 잭팟 풀이 50,000 크레딧 이상 누적된 상태이다.
2. 현재 필드에 잭팟 블록이 존재하지 않는다.
3. 새 구역이 생성될 때 각 블록 위치마다 **0.01% 확률**로 잭팟 블록 등장 여부를 판정한다.
4. 확률에 당첨된 경우 잭팟 블록이 해당 위치에 배치된다.
5. 잭팟 블록이 등장하면 이후 현재 필드에 있는 모든 유저에게 즉시 등장 알림이 전송된다.

---

### [STORY 9-3] 잭팟 블록이 등장하면 전체 유저에게 현재 풀 금액을 포함한 알림 배너가 표시되어야 한다.

1. 잭팟 블록이 필드에 등장한다.
2. 현재 필드의 모든 유저 화면에 특별 배너가 표시된다:
   - 배너 내용: "🏆 Jackpot Block 등장! Pool: {현재 풀 금액} 크레딧 🏆"
   - 황금색 특별 스타일의 배너
3. 배너는 **4초** 후 자동으로 서서히 사라진다.
4. 유저들은 이 알림을 보고 잭팟 블록 위치로 곡괭이를 집중 투입할 수 있다.

---

### [STORY 9-4] 유저는 곡괭이로 잭팟 블록을 공략하여 체력을 감소시킬 수 있어야 한다.

1. 유저가 곡괭이를 구매하여 잭팟 블록 근처에 배치한다.
2. 곡괭이가 잭팟 블록에 충돌하면 해당 곡괭이의 공격력만큼 잭팟 블록의 체력이 감소한다.
   - 잭팟 블록 초기 체력: **1,500**
   - 파워 곡괭이(공격력 5) 1회 충돌 시: 체력 1,500 → 1,495
3. 여러 유저가 동시에 각자의 곡괭이로 공격하면 피해가 합산되어 빠르게 체력을 줄일 수 있다.
   - **10~15명이 동시 공략 시 약 30~50초 내에 파괴 가능**
4. 타격이 쌓일수록 블록 표면에 균열이 점진적으로 증가하여 시각적으로 확인된다.
5. 체력이 남아 있는 한 잭팟 블록은 파괴되지 않고 게임이 계속 진행된다.

---

### [STORY 9-5] TNT로 잭팟 블록을 공격해도 40% 피해만 적용되어야 한다.

1. 유저가 TNT를 구매하여 잭팟 블록 방향으로 투하한다.
2. TNT가 잭팟 블록 근처에서 폭발한다.
3. 잭팟 블록은 TNT에 대한 저항력이 있어 폭발 피해의 **40%만** 적용된다.
   - TNT 기본 공격력 30 → 잭팟 블록에 실제 적용되는 피해: 12
4. 같은 폭발 범위 안에 있는 일반 블록(저항 없음)은 TNT 피해 30이 그대로 적용된다.
5. TNT와 곡괭이를 조합하는 전략이 효과적이다.

---

### [STORY 9-6] 유저가 잭팟 블록의 마지막 체력을 파괴하면 누적 풀 전액을 단독 수령해야 한다.

1. 여러 유저가 협력하여 잭팟 블록의 체력을 거의 소진시킨다.
2. 어떤 유저의 곡괭이가 잭팟 블록에 마지막 타격을 가하여 체력이 0 이하가 된다.
3. **마지막 타격을 가한 곡괭이의 소유자**가 현재 잭팟 풀 전액을 단독 수령한다.
   - 보상은 즉시 해당 유저의 잔액에 적립됨
   - 이전에 공격에 기여한 다른 유저들에게는 보상이 분배되지 않음
   - 수령 후 잭팟 풀은 0으로 초기화되어 다시 적립 시작
4. 화면 상단의 유저 잔액이 즉시 풀 전액만큼 증가한다.

---

### [STORY 9-7] 잭팟 블록이 파괴되면 파괴자 이름과 수령 금액을 포함한 전체 알림 배너가 15초간 표시되어야 한다.

1. 잭팟 블록이 최종 파괴된다.
2. 현재 필드의 모든 유저 화면에 파괴 알림 배너가 표시된다:
   - 배너 내용: "[파괴자 이름] mined Jackpot Block! +{수령 금액}cr"
3. 배너는 **15초** 동안 화면에 유지된 후 자동으로 사라진다.
4. 파괴자 본인 화면에도 동일 배너가 표시된다.
5. 화면 오른쪽 잭팟 피드에도 항목이 추가된다.

---

### [STORY 9-8] 화면이 잭팟 블록 위치를 지나쳐버리면 잭팟 블록이 소멸되되 풀은 유지되어야 한다.

1. 잭팟 블록이 필드에 존재하는 상태에서 화면이 계속 아래로 내려간다.
2. 잭팟 블록이 위치한 구역이 화면 상단으로 완전히 벗어나면 해당 블록이 자동으로 소멸 처리된다.
3. **잭팟 풀은 사라지지 않고 그대로 유지된다.** 계속 하우스 수익이 적립되어 증가한다.
4. 소멸된 잭팟 블록은 체력이 완전히 회복된 상태로 다시 등장할 수 있다.
   - 재등장 확률은 최초 등장 확률(0.01%)의 1/10인 **0.001%** (극악 확률)
   - 풀 조건(50,000 크레딧 이상)은 여전히 충족되어야 재등장 가능

---

### [STORY 9-9] 잭팟 블록이 파괴된 후에는 풀이 0에서 다시 적립 시작되어야 한다.

1. 잭팟 블록이 파괴된다.
2. 잭팟 풀이 **0으로 초기화**된다.
3. 이후 하우스 수익 누적 → 마일스톤 도달 → 풀 적립 사이클이 다시 시작된다.
4. 풀이 다시 50,000 크레딧에 도달해야 다음 잭팟 등장 기회가 발생한다.

---

### [STORY 9-10] 유저는 현재 잭팟 풀 금액을 게임 화면에서 확인할 수 있어야 한다.

1. 게임 화면 특정 위치(예: 상단 바, HUD)에 현재 잭팟 풀 금액이 표시된다.
2. 잭팟 풀이 0이면 표시되지 않거나 비활성 상태로 표시된다.
3. 잭팟 풀이 50,000 크레딧 이상이면 활성화된 스타일(황금색 등)로 표시된다.
4. 수치는 실시간으로 갱신된다.

---

### 정책 요약

**잭팟 풀 적립**
| 이벤트 | 내용 |
|--------|------|
| 곡괭이 수명 만료 | 하우스 수익 = 구매가 - 획득 보상 집계 |
| 하우스 누적 수익 50,000cr 달성마다 | 잭팟 풀 +2,500cr |

**잭팟 블록 스펙**
| 항목 | 값 |
|------|-----|
| 체력 | 1,500 |
| 스폰 조건 | 풀 ≥ 50,000cr AND 0.01% 확률 |
| 재스폰 확률 (이탈 후) | 0.001% |
| TNT 저항 | 40% (30 피해 → 12 적용) |
| 보상 수령 방식 | 마지막 타격자 독식 |
| 풀 파괴 후 처리 | 풀 0 리셋 |
| 풀 이탈 후 처리 | 풀 유지, HP 회복 |
